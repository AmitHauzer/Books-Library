# Books-Library-flask
## Installation:
1.  Clone from Git.
2.  Create a venv.
3.  Install the requirements file.
4.  run wsgi.py


## Functions:
### Admin user:
* all books.
* view a book.
* delete books.
* add books.
* all users.
* change users' permissions.
* delete users.


### Customer user:
* all books.
* search books.
* view a book.
* loans books.
* show all the active loans.
* change password.


### Guest user:
* all books.
* search books.
* register.
* Login.


### DataBase:
* sqlite3
* The database doesn't appear in the gitignore file in the meantime.
